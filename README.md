# Инструкция для менеджера — лендинг «Заберіть винагороду»

Короткая страница-«спасибо», которая ведёт участника опроса в Telegram-бота и учитывает
переходы в аналитике. Не требует программиста для базовых правок — всё меняется в двух файлах.

---

## 1. Из чего состоит проект

| Файл         | Что это                                   | Нужно ли трогать           |
|--------------|-------------------------------------------|----------------------------|
| `index.html` | Сама страница (тексты, кнопка, аналитика) | Да — здесь все правки      |
| `styles.css` | Оформление (цвета, отступы)               | Обычно нет                 |
| `logo.png`   | Логотип в шапке                           | По желанию (заменить файл) |
| `favicon.*`  | Иконка вкладки браузера                   | Нет                        |
| `robots.txt` | Запрет индексации поисковиками            | Нет                        |

> ⚠️ Все файлы всегда публикуются **вместе, в одной папке (в корне сайта)**.
> Если разнести их по разным папкам — пропадут стили, иконка и аналитика.

---

## 2. Что можно менять самому

### 2.1. Ссылка на Telegram-бота
Сейчас кнопка ведёт на `https://telegram.me/Discovery_survey_bot`.
Ссылка указана в файле `index.html` **в двух местах** — поменяйте в обоих:

1. В кнопке:
   ```html
   <a href="https://telegram.me/Discovery_survey_bot" ...>
   ```
2. В блоке аналитики внизу страницы:
   ```js
   var TELEGRAM_URL = 'https://telegram.me/Discovery_survey_bot';
   ```

### 2.2. Тексты
Заголовок, подзаголовок, пункты списка, суммы бонусов и текст кнопки — это обычный текст
в `index.html`. Найдите нужную фразу и замените её. Разметку (`<...>`) не удаляйте.

### 2.3. Логотип
Замените файл `logo.png` на свой (желательно похожего размера, с прозрачным фоном).
Имя файла оставьте прежним — `logo.png`.

## 3. ❗ Обязательный шаг: конверсия Google Ads

Аналитика Google Analytics (GA4) **уже работает** — переходы считаются автоматически.
А вот конверсия для **Google Ads** пока не «привязана»: нужно один раз вставить код конверсии.

**Где взять код:**
Google Ads → **Цели (Goals)** → **Конверсии** → выберите нужное действие →
раздел **«Настройка тега»**. Там будет значение вида:
```
AW-17667733049/AbC-dEfGh1jK…
```

**Куда вставить:**
В `index.html`, внизу страницы, найдите строку:
```js
var ADS_SEND_TO = 'AW-17667733049/CONVERSION_LABEL';
```
и замените `CONVERSION_LABEL` на часть после слэша из вашего кода.

До этого шага: переходы в GA4 считаются, а конверсии в Google Ads — нет.
Если не получается найти код — напишите разработчику, это займёт 1 минуту.

## 4. Как опубликовать изменения

1. Сохраните изменённые файлы.
2. Загрузите **все файлы папки** на хостинг (в корень сайта) — тем же способом, что и раньше
   (через хостинг-панель, FTP или того, кто занимается публикацией).
3. Откройте страницу в браузере и обновите её с очисткой кэша: **Ctrl + F5**.

---

## 5. Как проверить, что аналитика работает

1. Откройте опубликованную страницу.
2. В Google Analytics (GA4) зайдите в **Отчёты → В реальном времени** — вы должны увидеть себя
   как активного пользователя.
3. Нажмите кнопку «Перейти до Telegram-бота» — в списке событий появится `telegram_cta_click`.

---

## 6. Важно знать

- **Страница намеренно скрыта от поисковиков и ИИ-ботов** (Google, ChatGPT и т.п.).
  Это нормально: лендинг раздаётся по прямой ссылке, а не через поиск. На аналитику это не влияет.
- Ничего страшного, если случайно сохраните файл с ошибкой — просто загрузите предыдущую версию
  обратно. Поэтому перед правками полезно сохранять копию файла.
